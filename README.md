# Task 1: Sales & Demand Forecasting for Businesses

## Project Overview
This project builds a predictive model to forecast future business sales and demand based on historical data. It implements time-based feature engineering and regression techniques to assist businesses in supply chain and inventory management.

## Key Features
* **Feature Engineering:** Extracted numerical components (Year, Month, Day, DayOfWeek) from raw datetime datasets to capture seasonality and cyclical patterns.
* **Predictive Modeling:** Implemented a Linear Regression model to establish baseline trends and forecast future demand metrics.
* **Data Visualization:** Built visual charts using Matplotlib to plot historical actual sales against future forecasted trends for easy business interpretation.

## Skills Gained
* Time-series analysis
* Linear Regression modeling
* Business intelligence interpretation & evaluation
